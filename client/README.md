## How to Run the Project


2. Navigate to the project directory:
   ```bash
   cd .\client\
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

5. Open your browser and go to:
   ```
   http://localhost:3000
   ```

The feed UI should now be visible and functional.
